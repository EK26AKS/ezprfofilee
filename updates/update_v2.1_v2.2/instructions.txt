Update Instructions
-------------------
Download latest script from Codecanyon go to > updates folder and upload 3 zip file in below directions


application.zip  > upload path: Goto your project root folder > application   upload this zip file inside this folder and extract it.

assets.zip  > upload path: Goto your project root folder > upload this zip file on there and extract it.

config.zip > upload path: Goto your project root folder > application > config upload this zip file inside this folder and extract it.
